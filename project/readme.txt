Dataset Analysis:

Follow the following steps to run:

1. mkdir data plot && cd data
2. Download Health_and_Personal_Care and Video_Games from https://jmcauley.ucsd.edu/data/amazon/ into data
3. cd ../code && python main.py

NOTICE: install any missing library if asked
Arguments:
--random_seed  # all the experiments are conducted under default setting, but you can still change it

The resulting plots will be outputed into directory plot, the pos tagging result will be outputed into code/out.txt.
Other information is outputed into the consola.
